- **Top decile EU benchmarks** sur le base salary.
- **Equity généreuse** : ESOP de 22% à la Series A. Cible 25% post-Series B.
- **Transparence** : grille publique en interne, fourchettes affichées dans chaque offre.
- **Annual review** : juillet 2026, juillet 2027, etc. Pas de surprise.

## 4. Hiring process

### 4.1 Pipeline target

| Étape | Volume mensuel cible (mai 2026) |
|---|---|
| Sourced (inbound + outbound) | 1 200 |
| Phone screen | 280 |
| Technical / craft round | 130 |
| On-site (4-6h) | 60 |
| Reference checks | 28 |
| Offers extended | 22 |
| Offers accepted | 18 |
| Started | 15 |

**Funnel conversion target** : 1 200 → 15 = ~1,3% (industry standard 0,3-0,8%).

### 4.2 Étapes par fonction

**Engineering (8 jours médian)**
1. Recruiter screen (30 min)
2. Hiring manager (45 min)
3. Take-home (4-6h, paid 200€) OU live coding (90 min)
4. System design + 2 craft interviews + values fit (4h on-site)
5. Founder/exec final (45 min) — sur les hires Senior+
6. References + offer

**Sales (6 jours médian)**
1. Recruiter screen (30 min)
2. CRO/sales manager (45 min)
3. Mock pitch / discovery (60 min)
4. Panel : Sales + CSM + AE (90 min)
5. Founder final (30 min)
6. References + offer

**Design (7 jours médian)**
1. Recruiter screen
2. CPO portfolio review (60 min)
3. Design exercise (paid 300€, 4h)
4. Critique session + craft + values (3h on-site)
5. Founder final
6. References + offer

### 4.3 Values fit interview (60 min, obligatoire)

Conduite par 1 hiring manager + 1 IC. Questions :

1. *"Raconte-moi une situation où tu as dit non à ton patron."*
2. *"Quel est le projet dont tu es le plus fier — et qui n'a marché qu'à moitié ?"*
3. *"Quelle est l'erreur professionnelle qui t'a le plus appris ?"*
4. *"Qu'est-ce qu'on ne devrait jamais faire chez Axion, selon toi ?"*
5. *"Quelle est la chose que tu fais que personne ne te demande de faire ?"*

Évalue : honnêteté, agency, leçons internalisées, intégrité, drive.

### 4.4 References

- 3 références minimum : 1 manager direct passé, 1 collègue peer, 1 personne managée
- Appels conduits par hiring manager + recruiter
- Questions standardisées + champ libre

## 5. Onboarding

### 5.1 Pre-start (J-30 à J0)

- Welcome pack physique (livre Axion, swag, machine pré-configurée)
- Email de Maxime ou co-founder + voice note personnalisée
- Accès Slack en lecture seule J-7
- Premier projet préparé par le manager

### 5.2 J1 - J30

- **Day 1** : Welcome breakfast avec un fondateur, setup matos, intro buddy
- **Week 1** : Orientation (manifesto, principes, brand, security, compliance)
- **Week 2-3** : Premier projet livrable (typiquement small fix end-to-end)
- **Week 4** : Présentation devant l'équipe, NPS J30

### 5.3 30/60/90

- J30 : 1:1 avec manager, "qu'est-ce qui t'a surpris ?"
- J60 : 1:1 avec skip-level, ownership élargi
- J90 : Review formelle, calibration, premier vrai goal

### 5.4 KPIs onboarding

| Métrique | Target |
|---|---|
| Time to first commit (Eng) | < 5 jours |
| Time to first deal call (Sales) | < 15 jours |
| Time to first ticket closed (CS) | < 7 jours |
| Onboarding NPS | > 70 |
| Voluntary attrition année 1 | < 8% |

## 6. Diversité & inclusion

### 6.1 Goals 2026

- 38% femmes en Engineering (vs 22% baseline 2025)
- 50% femmes en Leadership (déjà atteint : 5/8)
- 25% under-represented minorities (sous critères locaux EU/US)
- Pay gap < 1,5% (audit annuel par cabinet externe)

### 6.2 Mécanismes

- Job descriptions revues par Textio (bias detection)
- Sourcing channels diversifiés : 50inTech, AfroTech, Ada Tech School, Lesbians Who Tech
- Diverse hiring panels (≥1 personne de groupe sous-représenté par panel)
- Mentoring programs internes

### 6.3 Bénéfices universels

- 26 semaines de congé parental rémunéré 100% (tous parents)
- Couverture santé + mutuelle famille premium
- 4 semaines de sabbatique tous les 4 ans
- Education budget 3 000€/an
- Mental health : 12 sessions thérapeute/an payées (Lyon Health, Spring Health)

## 8. Risques RH principaux

| Risque | Mitigation |
|---|---|
| War for AI talent | Top decile comp + brand fort + mission claire |
| Croissance trop rapide → culture diluée | Onboarding structuré + "values fit" rigoureux + 1:1 founders/leadership monthly |

| Leadership gap au scaling | VPs hires anticipated avec 2 quarters d'avance |
| Compliance EU/US (visa, IRS, etc.) | Cabinet Mercer + Deel + GlobalUpside dès recrut. cross-border |
